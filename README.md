# 좋은날금박 원격 PC 부팅 시스템

24시간 작동하는 WOL (Wake on LAN) 원격 부팅 서버

## 특징

- 70대 어르신용 큰 글씨 (32px 입력창, 48px 제목)
- 좋은날금박 녹색 테마
- 모바일 최적화
- 24시간 클라우드 서버

## 배포

Render.com 무료 배포:
1. Build Command: `pip install -r requirements.txt`
2. Start Command: `python wol_server.py`

## 사용법

1. 웹 접속
2. PC 등록 (MAC 주소, IP)
3. 버튼 클릭으로 원격 부팅
